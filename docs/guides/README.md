# Посібники користувача

Інструкції для кінцевих користувачів SattelishMaps.

## Зміст

- [Використання карти](using-map.md) - Як працювати з інтерактивною картою
- [Робота з фільтрами](filters-guide.md) - Фільтрація даних
- [Інтерпретація даних](data-interpretation.md) - Розуміння індексів NDVI, NDWI, NDBI, Moisture

## Швидкий старт для користувачів

1. Відкрийте http://localhost:3000
2. Виберіть регіон на карті
3. Застосуйте фільтри (дата, хмарність)
4. Перегляньте статистику

## Основні функції

### Перегляд карти
- Масштабування (scroll/pinch)
- Переміщення (drag)
- Вибір регіону (click)

### Фільтрація
- За датою
- За хмарністю
- За індексами

### Візуалізація
- NDVI (рослинність)
- NDWI (вода)
- NDBI (забудова)
- Moisture (вологість)
