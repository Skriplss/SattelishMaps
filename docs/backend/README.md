# Backend Documentation

FastAPI backend для SattelishMaps.

## Структура

```
backend/
├── api/              # API endpoints
├── services/         # Business logic
├── models/           # Pydantic models
├── utils/            # Utilities
├── config/           # Configuration
├── scheduler.py      # APScheduler
└── app.py           # FastAPI app
```
